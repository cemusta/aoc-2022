## Running locally

Ran and tested with node v16, start by installing packages with

```bash
# install packages
npm install
# install ts-node globally
npm i ts-node -g
```

You can run solutions with

```bash
ts-node ./src/day1.solve.ts
```

You can start tests with

```bash
npm test
```
